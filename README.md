Section A — Intro to Backend (Theory: 12 Questions)
What is backend development and how is it different from frontend development?
Ans: Backend is the server side of website development, it is deals with handling, processing, storing and retrieving of information and data.
It is different from Frontend because it works behind the scene unlike the frontend that users see and operate on
Mention three backend programming languages.
Ans: Python, PhP, C++
What is a server? Explain its role in backend development.
Ans: A server is a device or a software structured to process requests sent over by several networks (user).
Backend development handles the server-side operations, including processing data, managing application logic, and handling user requests.
Define an API and explain its purpose.
Ans: Application Programming Interface (API) is a part of development software that enables applications communicate and excharge data, its purpose is to communicate and exchange data with the frontend
What is a database and why is it important in backend systems?
Ans: Database is a system structured to collect, process, store and retrieve data electronically..it is important because data can be retrieved, data is structured, it is secured
List two differences between SQL and NoSQL databases.
Ans: SQL are structured to be in Table (having columns and rows) while NoSQL has a key value non relational model pattern
SQL is  Vertically scalable (upgrading hardware) while NoSQL is Horizontally scalable (adding more servers)
What is a backend framework? Mention one example.
Ans: Is the collection of necessary tools, modules, libraries and dependencies from a programming language essential to build a server side of a website application.
E.g Django (Python)
Explain what HTTP is and why it is important.
Ans: HTTP (Hypertext Transfer Protocol) is a fundamental protocol of the Internet, enabling the transfer of data between a client and a server. It is important because it enables exchange of information from the client and the server.
Mention two responsibilities of a backend developer.
Ans: Create APIs and Authentications,, Manage Databases and Logic
What does CRUD stand for? Explain each word.
Ans: C-Create(add new item or data), R- Read (view list or existing data), U- Update( edit existing data), D- Delete (delete data)
What is authentication and why is it important for backend applications?
Ans: Authentication is the process by which users are identified before having access to an application or a given page or information.
It helps protects data, restrict access to given information,
What is the difference between a GET request and a POST request?
Ans: GET: is used to retrieve data while POST: is used to send new data, which alters the database unlike the GET
